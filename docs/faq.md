# FAQs

For full documentation visit [mkdocs.org](https://www.mkdocs.org).

??? note "What is Material for MkDocs?"
    Material for MkDocs is a theme built on top of MkDocs that transforms documentation sites with a modern, responsive design inspired by Google's Material Design principles.

??? note "Is Material for MkDocs free?"
    Yes, the core theme is free and open source. All features, previously exclusive to the Insiders version, are now available to everyone.
