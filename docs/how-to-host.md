# How to Create an Event in Here & There

## 1) Create the Event
- Click the **Events** tab  
- Select **Create Event**

## 2) Complete the Basic Event Information

### a. Event Name
Keep the title **short, clear, and engaging**, and **always include the city name**.

Including the city helps members quickly identify where an event is taking place when browsing the event list on mobile or desktop. Not everyone will recognize a venue name, band name, movie title, or local landmark without location context.

**Examples:**  
- Bowling & Fun – Novi  
- Movies – Lansing  
- Live Band & Drinks – Flint   
- Cards & Games – Kalamazoo  
- Roller Skating – Warren

**Important:** The city name is required and should appear early enough in the title to be visible in event listings.

### b. Date and Time
Enter the correct date and time.

If there is a specific entry time or staggered arrival, use the intended arrival time and clarify in the description when the event officially begins.

### c. Address
Always include the full address.

This allows members to click directly into map navigation.

### d. Who Can See It
This should always be set to **Here & There Michigan**.

### e. Group Event Privacy
Set privacy to **Members Only**  
(Do not make the event public.)

## 3) Add Event Details

### a. Be Specific About Location
Remember this is a statewide group, and many members may not know local nicknames for venues, parks, or gathering spots.

For example, saying **"Montrose Pub"** may not help someone unfamiliar with the area unless the town is also identified.

### b. Help Members Feel Comfortable Attending
Include details that make the event inviting:
- Mention if a table or area is reserved  
- Let members know how to recognize the group  
- Post a photo once you arrive  
- Describe what to look for  

If someone offers to serve as a wingman, meet people outside, or help new members walk in, that can make a big difference—especially for newer, shy, or introverted members.

### c. Include Helpful Logistics
If applicable, mention:
- Costs  
- Ticket requirements  
- Limited seating  
- Parking details  
- Reservation needs  
- Anything useful for someone attending for the first time  

## 4) Add a Co-Host
You may add one co-host after the event is created (desktop is often easiest for this).

If it is your first time hosting, feel free to ask someone experienced to help.

## 5) Add an Event Banner

### a. Edit the Event Photo
Click **Edit** on the top banner image.

### b. Choose a Photo That Matches the Activity
Use an image that visually represents the event.

### c. Avoid Screenshots
Please avoid screenshots of flyers or other events that contain text, dates, or unrelated details.

Better options include:
- A venue photo  
- A band image  
- A hiking trail  
- Bowling lanes  
- Roller skates  
- A generic activity image  

A strong photo helps your event stand out and makes it easier for members to find later.

## 6) Create the Event
Click **Create Event** — and you're done 🎉

# Additional Notes

- Once created, the event automatically appears in the Here & There feed, so there is no need to immediately repost it.
- Promote your event thoughtfully. Sharing about once per week is usually enough at first, then increase reminders as the date approaches.
- Add fresh text when reposting rather than simply resharing without context.
- Encourage conversation directly on the event page—welcoming attendees and responding to comments helps visibility through group algorithms.

# What You Need as an Event Host

## Name Tags
Simple sticker name tags and a marker work perfectly.

## A Visible Marker
Having something noticeable such as a balloon, table sign, or distinctive clothing helps members identify the group easily.

# Why Visibility Matters

A visible, welcoming group naturally attracts attention—especially when everyone is engaged, laughing, and enjoying themselves.

It also helps first-time attendees, shy members, introverts, and those attending alone feel confident that they have found the right group.

# Important Reminder
**Check your event regularly.**

Members often ask questions before an event, and timely responses help attendance and improve the overall experience.
